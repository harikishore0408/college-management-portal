# College Managemnet Portal (MERN Stack) App

**Front-End** - React </br>
**Back-End** - Node.js, Express.js & MongoDB

## SetUp

### Prerequisite
 Must have node installed
### For server side </br>
- Clone the server folder and install the following dependecies
  * body-parser
  * express
  * connect-mongo
  * cookie-parser
  * cors
  * express-session
  * mongoose
  * multer
  * passport
  * passport-local
  * nodemon
- to start the server
  > npm start
- server location 
  > http://localhost:8000

### For client side
- First create react app using command 
  > create-react-app my_app
- Clone the client folder and install the following depedencies
   * axios
- to fire up react app 
  > npm start
- server location 
  > http://localhost:3000
